Problem Statement:

The Intermoutain Region recreation page does not have content useful for the public looking for information for their next trip.

Solution:

Implement an interactive map, recreational cabin database, links to forest web sites, and more present link to recreation.com.

Link:

Prioritization / Hierarchy

Using the information gathered, I could determine what content show be displayed on page and the importance.

1. Interactive Vistor Map

New as of early 2016 this map allows users to scroll over the forest they want to select recreation spots and get more information. This data is pulled directly from the forests' databases.

Buttons above change the location of the map to quickly jump to a forest.

2. Links To Forest Sites

The forests have all the recreation data, this drop down sends you directly to the forest's recreation page

3. Cabin Rental Database

I created a database of rentable cabins in the Intermountain Region, to allow users to find and rent these cabins.

The cabin information is on the forest website and the reservation link is on recreation.gov.

4. Links To Most Common Click Throughs

Google analytics showed that the most popular click though pages where maps and publications and passes and permits.
